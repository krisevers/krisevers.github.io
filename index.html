GIF Model

The generalized integrate-and-fire neuron model can be fit to a wide range of electrophysiological data from a number of species. Besides the integrate-and-fire mechanism the model also has a adaptive threshold mechanism instead of a fixed threshold. This makes the model very flexible when trying to data. The adaptive threshold mechanism can be interpreted as the neuron's spike-frequency adaptation (SFA) or sub-threshold adaptation (STA) dynamics. In the following I will go through the mathematical description of the model and provide a python implementation of the model as well as a simple random balanced network of GIF neurons.
